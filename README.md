Zephyr-RTOS module for APM32 MCU

# Compatibility

Supported zephyr versions:

* 3.4.0
* 4.1.0-rc1

## Supported hardware
- Board
  - apm32f103vc_mini
  - apm32f411vc_mini
- HAL
  - apm32f10x
  - apm32f40x
  - apm32f41x
- Driver
  - clock_control
  - flash
  - gpio
  - pinctrl
  - serial
